# Contributing to Matrix-iMessage Bridge

Thank you for your interest in contributing to the Matrix-iMessage Bridge project!

## Development Setup

1. Clone the repository
2. Install Rust (1.75 or later)
3. Copy `config/config.sample.yaml` to `config/config.yaml`
4. Update the configuration with your settings
5. Run `cargo build` to build the project

## Code Style

- Follow Rust standard formatting conventions
- Run `cargo fmt` before committing
- Run `cargo clippy` and fix any warnings
- Add tests for new functionality

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Ensure all tests pass
5. Submit a pull request
